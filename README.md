## 부마위키
부마위키는 부산소프트웨어마이스터고등학교의 역사를 기록하는 위키입니다.  
학생, 선생님, 사건사고, 동아리 등의 문서에 대해 다룹니다.  
교내 학생이라면 누구나 문서를 편집하고 생성할 수 있습니다.
